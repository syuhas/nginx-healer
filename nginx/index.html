Welcome to the NGINX target testing server!!

If you see this page, your NGINX server is up and running.

This project was designed to test the capabilites of using Ansible, Prometheus and AWS to design a custom self-healing infrastructure.

If this server goes down at any point, the automation will kick in via Alert Manager and fire the webhook to detect the failure and fix the issue.
